## Primeira rodagem

Para começar a utilizar o sistema em seu ambiente de desenvolvimento rode:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhosnt:3000](http://localhost:3000) em seu navegador para ver o resultado.
